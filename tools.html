Verktyg

Cocona

Framework

Ett framework för att bygga konsolapplikationer som påminner om att bygga applikationer för ASP.NET Core.

Eisvogel

LaTeX template

Jag använder den här mallen när jag skriver rapporter. Det blir rent och snyggt och ser inte ut som varje LaTeX dokument som nånsin gjorts.

Homebrew

Package manager

En pakethanterare som hanterar beroenden automatiskt som både kan installera från okompilerad källkod men även färdigbyggda binärer.

Imagemagick

Image Manipulator

Ett oerhört kraftfullt kommandoradsprogram för att manipulera bilder.

Insomnia REST

REST client

Det vanliga är att använda Postman men jag tycker det för tungrott. Insomnia innehåller allt jag behöver och har ett tydligare gränssnitt.

Jetbrains

IDE

IntelliJ är deras mest kända produkt men de har IDE med samma gränssnitt för alla stora programspråk. Ända sen jag började med Java har jag varit fast och kommer inte byta till VS Code eller Visual Studio ifall ingen tvingar mig.

Luca cv

Markdown template

Ett pythonscript som konverterar en markdown-fil till ett cv beroende på vissa taggar.

Manning.com

Book publisher

Det här är det bokförlag jag tycker håller mest jämn kvalitet och har pred med börjat med deras prenumerations-service.

Pandoc

Document converter

Pandock är en schweizisk arme-kniv för dokumentformat. Själv använder jag den till 99% för att konvertera Markdown till pdf.

Panwriter

Markdown editor

PanWriter är en enkel markdown editor integrerad med pandoc så det går att skapa pdf:er från markdown inifrån editorn.

Pixelmator Pro

En foto-redigerare som kostar lite pengar så inte gratis som Gimp, men är långt ifrån lika dyr som Photoshop. Det är också ett program som har en bra avvägning mellan features utan att kännas för fullspäckat.

Spectre console

.NET bibliotek

Ett fantastiskt bibliotek för att göra konsol-applikationer som är vackra och färggranna.
